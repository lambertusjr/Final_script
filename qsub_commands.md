# qsub Commands

All combinations of datasets and models for `submit_RP.sh`.

## Elliptic

```bash
qsub -F "Elliptic GCN" submit_RP.sh
qsub -F "Elliptic GAT" submit_RP.sh
qsub -F "Elliptic GIN" submit_RP.sh
qsub -F "Elliptic MLP" submit_RP.sh
qsub -F "Elliptic XGB" submit_RP.sh
qsub -F "Elliptic RF" submit_RP.sh
qsub -F "Elliptic SVM" submit_RP.sh
```
